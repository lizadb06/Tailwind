15/12 was de laatste les om verder te werken aan onze taken.

Tailwind: Om te beginnen heb ik op de homepage de dropdown van de weken weggedaan aangezien deze hier nog niet nodig zijn. Vervolgens heb ik wat tekst toegevoegd met uitleg over wat het bevat.
Deze heb ik later nog aangepast wat betreft de lay-out. Hierna heb ik het spoor van de muis toegevoegd. Deze is grijs omdat mijn schermen ook grijs bevatten.

Ook heb ik de GitHub link veranderd in een gefixeerde GitHub knop links beneden. Als laatste is daarnaast een download knop om mijn gecomprimeerde PDF van mijn documentatie te downloaden.
Deze is net zoals de GitHub link gefixeerd.

Op de pagina van de weken heb ik maar twee aanpassingen gedaan. Ik heb de weken in de gefixeerde header geplaatst zodat deze altijd zichtbaar zijn en de gebruiker tijdens het scrollen kan checken in welke week hij zit. Ook heb ik ervoor gezorgd dat hij snel tussen de weken kan switchen. Als hij een specifieke week wilt bekijken, kan hij de dropdown gebruiken.

Als laatste heb ik ook bij de weken de GitHub link aangepast net zoals op de homepage.

Trein-app: Deze week viel me pas op dat ik fouten heb gemaakt met de bestemmingen. Op het perron- en wagonsherm heb ik Eijsden als eindbestemming gezet maar het is eigenlijk Voeren. Bij de ticket stond er "Voeren-Eijsden". Ook heb ik vertrek- en aankomst tijd uiteen geschreven. Deze fouten heb ik gecorrigeerd.
